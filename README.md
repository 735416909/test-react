# test-react

###技术栈：
```
react + less + webpack + ES6 + flex.css + nodejs + fetch
```

### 实现的功能
```
  1.搜索数据（本地搜索，未请求服务器）
  2.修改数据
  3.删除数据
  4.添加数据
```

### 开发环境
```
  nodejs 6.3.0
  npm 3.10.3
  webpack 1.13.0
  开发的系统：mac
  开发的工具： vs code
```

### 运行
```
  npm install 安装依赖模块
  npm run dev 开发环境 访问：http://localhost:3000/dist/index.html
  npm run dist 生产环境编译
```

### 总结
```
  对比vuejs，react入门门槛比较高，
  但是对逻辑的处理能力要比vue强很多,同时也导致复杂很多。
```
